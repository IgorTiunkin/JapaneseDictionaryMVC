Квиз

В квизе вам предстоит по предложенному заданию выбрать один из правильных ответов.
Доступны два варианта квиза, выбрать которые можно, нажав соответствующую кнопку.
- "Иероглифы" : по записи на японском языке нужно определить перевод на русский.
- "Перевод" : по записи на русском языке нужно определить перевод на японский.

Введите количество заданий (не более 20)

Введите количество вариантов (от 2 до 10)
