Словарь

Записи в данном словаре содержат несколько полей, по каждому из которых можно искать. Вы можете выбрать поле поиска самостоятельно, отметив одну из опций под строкой для ввода текста. Подробнее о настройках поиска можно прочесть внизу.


В каждой записи есть следующие поля, которые будут выводиться при любом запросе:

Пример:
Иероглифы: 病
Кана: やまい
Перевод: болезнь, дурная привычка
Транслитерация: yamai
Если вы хотите целенаправленно искать по конкретному полю, отметьте соответствующую опцию.

Доступен поиск по умолчанию.
Этот поиск будет определять поле по следующему принципу:

Запрос на кириллице - поиск по переводу.
Запрос на латинице - поиск по транслитерации.
Запрос с использованием иероглифов - поиск по иероглифам.
Запрос с использованием каны - поиск по кане.
Не смешивайте разные языки (за исключением иероглифов и каны).

Также доступна опция "Только полные совпадения".
Правила определения:

Для иероглифов и каны будет выдана только запись с данным словом.
Для транслитерации будут выданы записи, где данное слово встречается изолированно.
Для перевода на русский язык будут выданы записи, где данное слово встречается изолированно именно в запрошенной форме.
