Приветствуем в нашем мини-словаре

Вы можете сами выбрать, по какому из полей искать или воспользоваться автоматической настройкой поиска

Автоматическая настрока будет определять поле по следующему принципу:
Запрос на русском языке - поиск по переводу.
Запрос на английском языке - поиск по транслитерации.
Запрос с использованием иероглифов - поиск по иероглифам.
Запрос с использованием каны - поиск по транслитерации на кану.
